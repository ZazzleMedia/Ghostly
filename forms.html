Input Append / Prepend

An input Append / Prepend is where you can group and add buttons and icons to join them together to form a unified single looking UI Element.

They work by looking for an .add-on class that will then change the look to work accordingly to which classes you've wrapped it in (input-append or input-prepend).

Input append / prepend also allows you to define a width for both the input and the add-on by providing how many columns you want it to be in brackets.

.input-append(@inputCols,@addOnCols) and vice versa for .input-prepend(@addOnCols, @inputCols). This then explicitly sets the widhts of them so that they will proportionally fill the element they are contained in. If nothing is set for .input-append() and .input-prepend() then the widths are auto.
